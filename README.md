# snakegame
So this started as a game of tic tac toe (which I wanted to build to make sure I actually understood python basics), then I built a bot that wins tic tac toe, then I thought if I could automate this, maybe I could build something bigger. This lead to the idea of building the snake game i used to play on my moms flip phone when i was a kid, in python. I chose this game because I thought it was elementary enough while still being cool. I didn't actually know if it was possible but I started with some research I found a tutorial, understood python structures, and iterated until I made it work. To be honest, I got stuck a couple times with tutorials so in my execution I ended up using methods I found in like 4 different places (partially because I didn't fully understand it the first time) + a little research but it worked by the end and I lwk found it so cool to be able to play something I made. I think this is the most tangible of the projects I've made so far. I think in the future maybe I could add colours or better sprites, or figure out how to do animation to actually make real games. 

To run this game simply save it to your computer and run that file through your terminal and it shold open right up. You can use the arrow keys to navigate and your score goes up when you eat an apple (an and). Your snake wraps when you hit a wall but if you hit yourself the game is over. And that's basically it.
